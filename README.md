2. zeromq
  * To get it running on Mac OS X (Yosemite):-
  * `$ wget http://download.zeromq.org/zeromq-4.1.0-rc1.tar.gz`
  * `$ tar xvf zeromq-4.1.0-rc1.tar.gz`
  * `$ cd zeromq-4.1.0`
  * `$ ./configure`
  * `$ make`
  * `$ sudo make install`
